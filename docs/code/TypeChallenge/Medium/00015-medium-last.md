# 最后一个元素
---

> 由谷歌自动翻译，欢迎 PR 改进翻译质量。

>在此挑战中建议使用TypeScript 4.0

实现一个通用`Last<T>`，它接受一个数组`T`并返回其最后一个元素的类型。

例如

```ts
type arr1 = ['a', 'b', 'c']
type arr2 = [3, 2, 1]

type tail1 = Last<arr1> // expected to be 'c'
type tail2 = Last<arr2> // expected to be 1
```
